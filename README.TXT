### Документация по проекту

# Что эта за игра?
Это не совсем игра, а полноценный телеграмм бот, в котором есть мини-игра жанра RNG.
RNG - это жанр игры в котором всё зависит от рандома.

# Как поиграть?
Чтобы поиграть нужно для начала установить библиотеки 'telegram' и 'asyncio'.
После установки запустите файл 'main.py'.
И уже после этого можете писать боту '@rng_game_821_bot' в телеграмме

# Доп информация:

Автоматы с казино не имеют смысл, просто автора били воздухом в детстве.
